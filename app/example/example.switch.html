Switches

Switches are enhanced radio buttons and checkboxes, and behave just as those inputs do.
Each switch can connect to the Angular data model ngModel. so like standard inputs, you can bind with [(ngModel)]="data"

For inner labels, add a div with attribute activeText for the 'on' position, and another with attribute inActiveText for the 'off'.

Inputs:

  • switch-id : Required id
  • name : Required name (as per radio group name) for radio switches, and required for checks if ngModel is used
  • size : "tiny", "small", "large" or default normal/standard size
  • title : screen reader-only text
  • value : for radio switches selected value
